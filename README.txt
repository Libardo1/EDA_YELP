Please follow the instructions to run our project using RStudio.

1. Download our chosen data set from the following link.
http://www.yelp.com/dataset_challenge/

2. Open RStudio. Click open project and select RScriptProject1.R file.
This opens and run our project and creates the workspace.

3. All the plots can be found under Plots tab which can be navigated using left and right arrows
